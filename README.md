## Dotfiles
Installation using stow

Ubuntu/Debian
```
sudo apt install stow
```
```
git clone git@github.com:Imabanana80/dotfiles.git ~/.dotfiles
```
```
cd ~/.dotfiles || stow .
```

### Software
These dotfiles contain configurations for the following:
```
zsh
nvim (oh-my-zsh)
git
neofetch
tmux (tpm)
```
