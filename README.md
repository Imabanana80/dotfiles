## Dotfiles
Installation using stow

Ubuntu/Debian
```
sudo apt install stow
```

AUR (Arch/EndeavourOS)
```
yay -S stow
```

Clone repo
```
git clone git@knot.potassium.sh:banana.tngl.sh/dotfiles
```
Run stow
```
cd ~/.dotfiles || stow .
```
